# CarDekho Sales Dashboard

## Project Overview
Analyzed used car sales data from CarDekho to identify trends in pricing, fuel type, ownership, transmission, and seller type.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
- Seaborn 

## Dataset
CarDekho Used Car Dataset

## Key Insights
- Diesel cars have higher average selling prices.
- Most listed cars are manual transmission.
- First-owner vehicles dominate the dataset.
